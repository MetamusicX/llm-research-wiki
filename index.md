## How to Navigate

The index is organized in four layers:
1. **Concept clusters** — thematic groupings for fast query navigation; check these first
2. **Authors** — all author pages, grouped by area
3. **Everything else** — debates, syntheses, source notes, projects
4. **Log** — change history in `log.md`

**Query protocol:** identify the relevant cluster(s) first, then read the synthesis page if one exists, then read the specific concept/author pages indicated. Do not scan the full index on every query.
